# 番茄短篇虐心婚恋大纲

## 工作流程

1. 先确认目标产物：`选题方向`、`剧情节奏分析`、`全文大纲`、`开篇钩子`、`分段细纲` 中选择一种；如果用户已经说清楚，就直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-short-nuexin-patterns.md`，把输出锚定在本地短篇虐心婚恋样本的结构，而不是泛泛写虐文。
3. 先定一句话核心伤害：谁在什么关系里，用什么理由践踏了谁，造成了不可逆损失。
4. 再定反击/惩罚方式：离开、死亡、真相公开、舆论反噬、法律审判、精神崩溃、事业败落、永失所爱，只选最能兑现情绪的一条主线。
5. 制作大纲时按短篇分段推进，不要写成长篇卷纲；每一段都要有明确情绪功能：扎心、补刀、逃离、真相、追悔、惩罚、余韵。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 优先输出可直接写正文的内容，不写空泛理论。
- 开篇必须在 300 字内抛出强钩子：死亡、流产、离婚、背叛、尸体、孩子受辱、被抛弃、替身真相等。
- 虐点必须具体到动作和证据，不只写“男主误会女主”：要写离婚协议、冷落记录、聊天证据、验尸报告、死亡证明、孩子伤痕、法院/医院/警方材料。
- 女主不要只靠哭获得爽感；要让她至少拥有一种能完成情绪闭环的力量：清醒离开、事业能力、证据、朋友、孩子、社会舆论、法律流程、死亡后的旁观视角。
- 男主后悔不能只写“痛哭”；要有代价：失去抚养权、精神崩溃、舆论反噬、财产转移无效、白月光反咬、亲友决裂、余生赎罪无门。
- 不要照搬样本人名、设定、标题、标志性桥段或具体句子，只提取节奏机制。
