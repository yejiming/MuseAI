# 番茄玄幻脑洞大纲

## 工作流程

1. 先确认目标产物：在 `选题方向`、`核心设定`、`前30章大纲`、`长线卷纲`、`正文试写` 中选择一种；如果用户已经说清楚，就直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-xuanhuan-patterns.md`；分析样本时要结合每本 `目录.md` 的完整章节标题，不只看前 30 章正文。
3. 每个想法都要锚定一个清晰卖点：反常识金手指、高压开局身份，或读者熟悉的神话、游戏、国家机器、系统规则变体。
4. 围绕短爽点闭环搭大纲：第1章亮钩子，第2-5章证明优势，第6-10章完成第一次反转，第11-20章扩大社会系统或地图，第21-30章收束第一场大冲突并打开下一张地图。
5. 输出可直接使用的写作产物，不写空泛理论；优先给出逐章事件、冲突来源、爽点兑现和章末悬念。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 每 3-5 章都要让主角目标可见。
- 制作大纲时使用具体章节标题。
- 每章都包含 `主要事件`、`爽点/钩子`、`章末悬念`。
- 除非用户明确要求，否则不要走文学慢热路线；番茄玄幻脑洞需要快速识别、快速优势、快速冲突和高频兑现。
- 不要照搬样本剧情、人名、章节标题或标志性措辞，只提取结构和机制。
