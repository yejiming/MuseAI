# 番茄玄幻脑洞样本规律

本参考资料来自本地 `番茄小说/玄幻脑洞` 下的 10 本番茄玄幻脑洞样本。每本正文已下载前 30 章，但 `目录.md` 保留了完整章节标题；分析样本大纲时要以后者推断全书阶段，不要只停留在前 30 章。

## 核心节奏

前30章通常不是“铺世界观”，而是连续证明卖点。

- 第1章：抛出一句话级卖点。常见形态是穿越/重生/长生/游戏提前登录/上交国家/反派系统/词条模拟/西游神境/妖魔图鉴。
- 第2-3章：立刻让金手指可见。读者必须知道主角为什么特殊，以及这个特殊会怎样换来资源、实力、身份或爽感。
- 第4-10章：第一轮闭环。主角用优势解决一个小困境，拿到第一批战力、身份、功法、组织信任或信息差。
- 第11-20章：扩大舞台。把个人优势接入宗门、朝堂、国家、游戏榜单、妖患、秘境、神话副本等外部系统。
- 第21-30章：完成第一场大冲突。通常是斩杀强敌、揭露骗局、转职升官、宗门震动、国家实验突破、地图回归或新身份曝光。
- 第30章末：必须开下一钩子。常见是新地图、新势力、家族隐秘、榜单变化、古殿秘密、国家项目升级、隐藏大佬现身。

## 共性爽点

- 信息差：主角知道剧情、游戏版本、模拟结果、神话真相、国家科技、未来走势。
- 身份差：表面杂役/矿工/小孩/赘婿/普通玩家，实际有系统、国家、长生、帝资、神话关系。
- 资源差：别人苦修，主角用词条、模拟、道行、长生点、国家实验、游戏机制、图鉴收录快速积累。
- 情绪差：外界质疑、轻视、追杀、交易压价，主角用结果打脸。
- 组织差：个人金手指和大组织绑定，形成“我不是一个人在开挂”的放大器。
- 长线差：短期解决麻烦，长期埋更大的命运/世界真相/版本更新/古老阴谋。

## 样本大纲

### 《一年抽取一词条，模拟的也可以？》

全书 385 章。卖点是词条抽取加人生模拟：主角用模拟先试错，再把最佳路线带回现实。

- 1-30章：神话词条和模拟器登场，主角通过多次模拟接触边军、家传功法、屠村线索和县衙冲突，完成“模拟试错-现实兑现”的基础闭环。
- 31-80章：万魂幡、古塔、赤心门、九州鼎、大道仙器等资源线展开，主角从个人求生转向截胡至宝、收服妖魂、卷入两族与正统之争。
- 81-130章：修为突破、沈家和灵海境线推进，乾坤镜、契合身体、古族与血脉身份不断抬高主角背景。
- 131-180章：两族大战、换卵事件、九州鼎围攻、妖族和人族正统冲突成为主线，主角用模拟和词条参与大势。
- 181-230章：七武神、心之钢、禁地和多件至宝连续登场，主角从局部战斗转入世界级资源争夺。
- 231-280章：逍遥神、武王突破、灵魂与战斗身躯、焚心不灭诀等线索，把个人成长推到高阶战力与神秘传承层。
- 281-330章：修改人界坐标、突破武圣、妖族离去、神话词条再开，主角开始影响世界坐标和种族格局。
- 331-385章：神兵启示录、矿镐造反、白粥起义、星舟星港航道、虚界和化丹突破，把玄幻升级推向星海/虚界舞台，并以救伙伴和“此间乐”收束当前阶段。

结构要点：目录显示它不是单纯模拟流，而是“模拟试错 -> 截胡神器 -> 种族战争 -> 星海/虚界扩张”。写同类大纲时，模拟必须不断产出新地图、新词条、新势力和现实兑现点。

### 《从变身少女开始斩妖除魔》

全书 755 章。卖点是百妖谱收录妖魔：主角以斩妖获得神通、身份和更大的妖魔世界入口。

- 1-30章：百妖谱、虎山神、狼妖、猪妖、龙王事件完成第一轮案件闭环，主角从村落求生走进镇魔司。
- 31-90章：入鸣骨、升官、先锋营、边关、长公主、成丹境等线索，把斩妖从县城案件推向军镇和边关体系。
- 91-150章：宝刹寺、平天真君、白猿公、义父、血肉兵装等势力交锋，主角开始面对宗教、军伍、妖族多方压力。
- 151-250章：姜洵之案、勾结妖魔、万妖吞天、点墨圆满等标题显示主线转为内部审判与更高层妖魔法门。
- 251-350章：血雨浴八荒、万化十四剑、五万年功力等标题，将战力层级推到大规模大战和高阶武学。
- 351-450章：五仙山危机、力战五仙、底蕴展示，主角开始进入山门级、仙山级冲突。
- 451-550章：画境、斩妖皇、登楼九重，案件单元升级为妖皇级资源和境界跃迁。
- 551-650章：道棋、界青宗盘算、各方布局出现，主线从单纯斩妖变成宗门棋局和高层博弈。
- 651-755章：第五尊道棋、界青亲传、力战高青休、玉京问道图、天庭若怒、三三化灵、天品出世，故事进入天庭/道图/天品资源线，并以“发达了”打开更高收益。

结构要点：案件流的每一案都要带来“妖魔收录 + 身份升级 + 新势力入口”。后期要从地方妖患升级到仙山、宗门、天庭级秩序。

### 《反派：我的弟弟是天选之子》

全书 2095 章，含番外。卖点是反派哥哥绑定天选弟弟和反派系统，用“秦老六”式操作薅气运、搅局、背锅和收割。

- 1-100章：天选弟弟、大帝之资、至尊骨、逐出秦家、大凶之地等开局事件，完成反派身份和老六人设定型。
- 101-300章：国法、公道、大局、专业大反派等标题显示主角进入王朝/宗门秩序冲突，用道德反转和舆论错位制造爽点。
- 301-500章：战神归来、宝藏、趁天选之子不在家、荒古总导演，主角开始把多个天选之子、宝藏和大势当成舞台调度。
- 501-700章：公子请留步、阅尽人间春色、唯恐天下不乱等标题强化喜剧、桃花和搅局支线，主角靠社交与资源盘活多线收益。
- 701-900章：剑意、昏君、放火、天子/皇权相关冲突，主角从家族反派扩张为影响王朝和正邪格局的人。
- 901-1100章：坎与命、不忘初心、菩萨心肠等标题持续反讽，主角在危局中用“看似缺德、实则获利”的方式维持节奏。
- 1101-1300章：卡点救场、油水、捆仙绳、赵长生等标题显示多角色同台，主角反复在救场、夺宝、背锅中滚雪球。
- 1301-1500章：盟主/萌主、玲珑道心、支援秦老六等线索，进入联盟和高阶道心/道统竞争。
- 1501-1700章：天机屠龙、易天机掀桌、坐标、黑锅等标题，把“老六操作”升级为天机局、屠龙局和因果局。
- 1701-1900章：大道挑战、神灵流血等标题显示战力和世界观升至大道/神灵层级。
- 1901-2095章：雷劫人选、天宫秘密、热身结束、番外专业团队，主线收束到天宫、高阶雷劫和团队番外。

结构要点：超长反派喜剧不能只靠开局梗，要不断换“天选之子/宝藏/王朝/天机/天宫”舞台，让主角每卷都能用不正经手段拿到正经收益。

### 《开局上交修仙界，国家让我先成仙》

全书 549 章。卖点是主角穿越修仙界后主动上交国家，国家机器成为最大外挂。

- 1-50章：报案、国家相信、跨界传讯、灵矿提纯、法甲炼器和宗门支线，完成“个人穿越变国家项目”的基础闭环。
- 51-110章：法甲攻防、灵根普查、修炼试点、大夏修身纲要、规章制度等标题，显示修仙开始制度化、工程化。
- 111-180章：长城选拔、跨界文明研究局、虚拟世界、资源争夺等线索，国家机器从科研转向军事和文明战略。
- 181-240章：青池不复、离天当立、并肩作战、道统转移，主角所在修仙界势力被大夏重新组织。
- 241-310章：真人授业、国运新章、道院、贡献天榜、离天道院清剿，蓝星修仙教育和异界作战体系成型。
- 311-380章：七道天命、王道龙气、天道、龙种和龙位命格，大夏开始触碰气运、命格和天道层规则。
- 381-440章：证道神君、属于自己的世界、蓝星气运监测、因果了结，主角和国家同步进入神君/世界主级别。
- 441-500章：深空战略物资、开新图、新躯生产、道廷弟子、西州各司，故事由单一异界扩张为深空和多地图战略。
- 501-549章：寂墟、鱼形玉佩、陆离回归、八阶神魂之宝、至尊仙种战、星空文明，主线推到异界星空竞争，并以响彻星空收束阶段目标。

结构要点：国家流要逐步升级组织能力：相信主角 -> 科研解析 -> 全民修炼 -> 跨界军政 -> 气运天道 -> 深空文明。

### 《开局长生万古，苟到天荒地老》

全书 2712 章。卖点是长生不死加点，主角和大黑牛以极度谨慎跨越凡间、修仙界、大时代、仙庭和纪元终局。

- 1-200章：长生、加点、凡间生活、仙门、炼丹、筑基、五蕴宗等线索，完成苟道种田和修仙入门。
- 201-400章：人体秘藏、五行仙道、丹鼎宗、界域毁灭、生死长生、秽寿小龟等标题，主线从小宗门推进到界域和长生本质。
- 401-600章：九天幻梦星枢、古族、岁月沉淀、仙尊之姿、大时代应运之人，陈浔从苟道小人物变成被大时代注视的长生者。
- 601-800章：神魄龟裂、大劫、塑仙躯、半仙、长生族谱、遂古鹤灵树等线索，进入半仙和长生族谱阶段。
- 801-1000章：天地死气、投胎、离开五蕴宗、梧城地摊、何为仙人，故事回到人间烟火，用低调生活消化高阶境界。
- 1001-1200章：狱象、作壁上观、道蕴成海、新生陈浔、鸿福散人，主角在更复杂势力中保持旁观与布局。
- 1201-1400章：仙庭历、万年、一门仙人、往生仙典、天道法纹、仙境第六步，主线进入仙庭和仙境体系。
- 1401-1600章：飞仙、长虹贯天、大道祥云、沧桑巨变，陈浔经历飞升和世界大变。
- 1601-1800章：抢婚、赏星河、晚风等标题显示高强度主线间穿插人情、故人和日常温度。
- 1801-2000章：恒古纪元、金丹洞天、驶向未来、与未来空气斗智斗勇，故事从仙界当前线推进到纪元未来线。
- 2001-2200章：生命跃迁、虚无残体、没有活人、刺杀极衍、炼九劫道，世界进入虚无、残体、纪元灾难阶段。
- 2201-2400章：仙界推举、证道洪潮、万道生莲、大道归一、道无止境，主角接近大道归一与仙界秩序核心。
- 2401-2600章：纪元天洞、因果自循、天地衰退、一条路走到黑，长生与纪元衰败正面相撞。
- 2601-2712章：无尽时代、最后一位、末法、万墓丛、长生烬，终局落在长生代价、时代耗尽和大结局余烬。

结构要点：长生超长篇靠“时间尺度变化”推进：凡间几十年、宗门数百年、仙庭万年、纪元无尽。每次升级都要有故人、旧地、时代变迁来承载情绪。

### 《我都大罗金仙了，你们怎么才武圣》

全书 376 章。卖点是现代神境副本连接西游，主角用道行、经典知识和方寸山关系改写西游节点。

- 1-50章：西游神境、方寸山、聊天群、听讲送道行、悟空和双叉岭，完成副本规则与方寸师兄弟羁绊。
- 51-100章：真武大帝、先天至宝、红尘历练、定箭倒射、法宝战等标题，主角从方寸修行转向外部神话资源争夺。
- 101-150章：缚龙索、小白龙、千宝道人、罗老师恩怨、戮魂幡等线索，现代探索者与西游龙族/法宝线交织。
- 151-200章：猴哥出师、千年道行、赠宝师兄、阎罗告状、玉帝震怒，主角开始直接牵动天庭和地府。
- 201-250章：十万天兵、四海龙王、四龙争婿、神风珠、彻底决裂，天庭/龙宫压力升级。
- 251-300章：火云洞、涂山氏、三头六臂、大乘登仙、敖烈刚鬣、沙悟净，主角提前重组西游取经团队资源。
- 301-350章：祖师归去、无垢先天道体、五方揭谛、佛国、南天门，故事由取经队资源扩张到佛门和天庭正面秩序。
- 351-376章：凌霄殿行走、仙官、人参果树、镇元子、地府作证、灵宝天尊点迷津、王母寻织女，后期进入天庭官职和大仙级因果局。

结构要点：神话副本流要把熟悉节点拆成“提前介入点”：方寸、龙宫、地府、天庭、火云洞、人参果，每个节点都要让主角拿到原著没有的收益。

### 《提前登陆游戏世界，开局联姻女帝》

全书 1406 章。卖点是提前登录游戏世界，主角绑定未来女帝/公主线，利用玩家经验、朝堂身份和战争大势抢先布局。

- 1-100章：提前登录、尚公主、羽林军、皇城司、三皇子、文会和天师府，完成朝堂身份、玩家优势和女帝线的起步。
- 101-250章：军团作战、天下大势、西域、重现神话，主角从京城权谋进入大规模军事和区域争霸。
- 251-450章：血液炼丹、皇宫、父慈女孝、扶持南疆、计划有变，朝堂家庭线与南疆战略线交织。
- 451-650章：禅让、登基、征服感，女帝/皇权线进入权力交接和主角辅佐登顶阶段。
- 651-850章：大黄牛、张苍叶修、擒龙、群龙齐聚、中原大决战，主线升级为龙族和中原决战。
- 851-1050章：阴阳圣经、大月皇宫、宣誓主权、争议，故事转向跨国皇朝与阴阳修行体系。
- 1051-1250章：蔚蓝色星球、剑哀、生死一线、阴阳蛟龙、天机老人再现，现实/星球线与天机线打开。
- 1251-1406章：对赌、七妙琉璃草、太荒锁、第一局、问剑红月城、夏辰以身入局，后期进入更高层博弈和以身入局的权谋终段。

结构要点：提前登录流的核心不是“知道剧情”四个字，而是持续抢占版本关键点：皇权、军团、龙族、外域、天机、赌局。

### 《无人知晓的百年，我成了大帝》

全书 256 章。卖点是主角表面入宗当杂役，系统觉醒大帝修为/超强资质，隐藏身份横扫同境，并逐步追查仙母与诸天隐秘。

- 31-70章：秘境、血煞宗、神秘女子、金丹异象、至尊狐王、青云弟子追随、全员灵根蜕变，主角从个人强大扩展到宗门整体升级。

- 151-200章：冰月宫主、玲珑圣地、洛清寒怀疑、化神丹、破圣丹、苏落霞突破半圣，情感线、圣地线和半圣战力并行。
- 201-230章：异象惊天下、推演、仙母踪迹、牛家村回溯，故事回到最初身世谜题，并通过推演扩大到诸天层。
- 231-256章：诡异小女孩、小幺幺、青铜面具、天降异象、诸天震动、大佬苏醒、诡异洞府，当前结尾以仙母最后踪迹和更高黑暗力量收束。

结构要点：短篇宗门爽文可用“宗门碾压 -> 秘境资源 -> 宗门整体升级 -> 身世推演 -> 诸天诡异”快速推高层级。

### 《系统赋我长生，我熬死了所有人》

全书 2348 章。卖点是长生系统让主角跨越时代，以送葬、沉睡、故人、布局和纪元秘密串起漫长世界。

- 1-200章：沉睡十年、仙人降临、人造灵根、棺材铺、青铜古殿、五百年布局和陈十三挥剑，建立长生者“见证与出手”的基调。
- 201-400章：苍鸿杀心、惊天一剑、雷劫技巧、前路断了、化凤未来、白泽等线索，主角开始介入强者道路断裂和未来修行体系。
- 401-600章：张百忍消失、牛头马面、动乱、四方大陆、天魂体系、觉醒天魂，世界扩到大陆远征和新修炼体系。
- 601-800章：弑神兵、太明天秘密、纪元渗透，主线转向禁地、神兵与纪元背后的隐秘入侵。
- 801-1000章：重回纪元、诡异、地龙石、纪元执念、净世咒和新火焰，主角在旧纪元和诡异力量之间寻找解决路径。
- 1001-1200章：纪元概念、虎贲、晚宴、水月掀桌、成见，世界观进入纪元政治和多方共识阶段。
- 1201-1400章：四大奇书、水书、家长、行万里路、天骄云集，主线变为知识体系、传承和天骄大会。
- 1401-1600章：黄金盛会、外公、蒙面杀戮、重新选择，人物关系与大会冲突推动情绪高潮。
- 1601-1800章：新的线索、初次交锋、载入史册战果、士为知己者死，主角面对更明确的敌对势力与历史级战果。
- 1801-2000章：李长生、刘一刀、帝师爱护、少年侠气、明行禅师，长生者继续用师长/帝师身份影响后辈命运。
- 2001-2200章：神兽肉身、追寻仙帝、空间乱流、众怒、至圣手稿，主线抬到仙帝线和至圣资源。
- 2201-2348章：帝君传道、坐等天下十万年、直面仙帝、碎道重修、迎战天骄、九华帝君三次打击，当前目录收束在仙帝压力与重修破局。

结构要点：长生送葬流要让每个时代有“新人主角”，而真正主角负责见证、点拨、布局和最后兜底。

## 后续选题建议

优先选择“熟悉题材 + 一个反常识外挂 + 组织/系统放大器”。

- 国家流：`我把宗门秘境上交后，全国开始修仙工业化`。卖点是国家科研、军工、教育体系解析修仙。
- 游戏流：`别人氪金抽卡，我在新手村修复漏洞成神`。卖点是底层规则漏洞和版本信息差。
- 长生流：`我每沉睡百年，醒来都能继承一个时代的债`。卖点是离别、旧账、历史布局。
- 神话副本流：`我在封神副本当群管理员，功德全被我结算了`。卖点是熟悉神话人物加现代规则。
- 反派流：`我是天命主角的账房先生，专门替他管理机缘`。卖点是反套路薅气运但不直接抢主角命格。
- 宗门隐藏流：`宗门以为我是守山杂役，其实护山大阵是我写的`。卖点是低身份高权限。
- 妖魔图鉴流：`每收录一种妖魔，我就能给人间打一个补丁`。卖点是图鉴、规则修复、案件单元。
- 模拟词条流：`我能模拟宗门百年兴亡，于是先把掌门培养歪了`。卖点是模拟未来和现实经营。
- 朝堂玄幻流：`我提前登录王朝版本，开局成了女帝的黑名单第一名`。卖点是未来女帝、权谋、玩家机制。
- 种田苟道流：`我在灵田里种了三百年，外面已经换了十个仙朝`。卖点是慢时间和资源碾压。

## 大纲制作要求

### 必填信息

- 一句话卖点：用一句话说明读者为什么点进来。
- 主角初始困境：身份低、资源少、被误解、被追杀、被迫选择或世界危险。
- 金手指规则：触发条件、收益形式、限制、升级方式。
- 第一章钩子：开篇3段内给出异常事件或强烈身份反差。
- 前30章目标：主角要拿到什么身份、资源、能力或组织入口。
- 第一反派/阻力：前30章必须有一个能被解决的具体对手或具体危机。
- 长线悬念：第30章后继续读的理由。

### 前30章模板

- 1章：异常开局，直接亮卖点。
- 2章：金手指或信息差可视化，主角做出第一选择。
- 3章：第一次小收益，外界质疑或危险逼近。
- 4-5章：主角验证规则，拿到第一件资源/身份/功法。
- 6-8章：第一场正面冲突，展示优势但留下更大麻烦。
- 9-10章：第一次爽点闭环，主角获得阶段性认可或收益。
- 11-13章：引入更大组织、地图、榜单、宗门、国家或副本。
- 14-16章：规则升级，主角拿到第二能力或关键情报。
- 17-20章：外部势力正式注意主角，身份/实力被部分看见。
- 21-24章：第一大冲突爆发，主角主动布局而非被动挨打。
- 25-27章：越级、反杀、揭露、转职、升官或打脸。
- 28-30章：收束第一事件，结算收益，抛出下一卷钩子。

### 检查清单

- 第1章是否能用标题和前3段说明卖点？
- 第5章前是否至少完成一次收益反馈？
- 第10章前是否有一次明确打脸或反转？
- 第20章前是否进入更大的社会系统？
- 第30章是否既完成一件事，又打开下一件事？
- 每3章是否有一个新信息、新收益、新敌人或新悬念？
- 主角是否一直在主动利用规则，而不是等系统发奖励？
- 金手指是否能重复制造剧情，而不是一次性解释完？
